# Stream.it Backend
Music Streaming app made in SEAN stack

This is the backend

* Uncomment to create a new user or DIY. Run ```create_DB.sql``` to setup the MySQL database

* Backend developed in 
   * NodeJS
   * ExpressJS
   * JWT Authorisation
   * MySQL
